# Simon Game 🎮

A fun and interactive memory game where you need to remember and repeat an increasingly longer sequence of colored buttons.

## Features ✨
- Responsive design - works on all devices
- Beautiful gradient UI with smooth animations
- Increasing difficulty levels
- Score tracking
- Sound feedback and visual effects

## How to Play 🎯
1. Press any key to start the game
2. Watch the colored buttons light up in a sequence
3. Click the buttons in the same order
4. Each level adds one more button to the sequence
5. Try to reach the highest level possible!

## Files
- `simon.html` - Game structure
- `simon.css` - Styling and responsive design
- `simon.js` - Game logic

## Deployment
This project is deployed on [Vercel](https://vercel.com)

## Technologies Used
- HTML5
- CSS3 (with gradients and animations)
- Vanilla JavaScript
